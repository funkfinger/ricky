# Google Coral Ideas

Jay has **2x Google Coral** USB Accelerators — edge TPUs for fast ML inference.

## What Coral Is Good At

- Real-time image classification
- Object detection
- Pose estimation
- On-device inference (no cloud needed)
- Low latency (~10ms inference)
- Low power consumption

## Project Ideas Using Coral

### Nature & Wildlife
- [[bird-oracle|Bird Oracle]] — Species ID + storytelling
- **Critter Cam** — Log all backyard wildlife
- **Insect identifier** — Macro camera + Coral

### Home Automation
- **Person vs motion** — Smart alerts only for humans
- **Package detection** — Know when deliveries arrive
- **Pet tracker** — Which pet is where
- **Visitor recognition** — Family vs stranger

### Practical/Maker
- **[[component-seer|Component Seer]]** — ID parts from Jay's inventory
- **Sorting assistant** — Help categorize/organize things
- **OCR station** — Read text from labels, components

### Art
- **Pose-reactive installation** — Art that responds to body position
- **[[ambient-life-counter|Ambient Life Counter]]** — Count everything that passes

## Available Models

- MobileNet (image classification)
- SSD MobileNet (object detection)
- PoseNet (human pose)
- Bird species classifier
- Custom models via TensorFlow Lite
